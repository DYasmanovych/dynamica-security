# Dynamica Security — Landing Page

Business-first cybersecurity consulting landing page. Built with pure HTML/CSS/JS — no frameworks, no build step.

## Stack
- Static HTML with embedded CSS and vanilla JS
- Google Fonts (JetBrains Mono + Outfit)
- Scroll-reveal animations via IntersectionObserver
- Deployed on Vercel

## Deploy
```bash
vercel --prod
```
